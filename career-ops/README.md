# Career-Ops 实习求职 AI 助手

> 模仿 [santifer/career-ops](https://github.com/santifer/career-ops)，专为中国市场实习求职定制

## 快速开始

### 第一步：完善您的简历
打开 `cv.md`，将所有 `[占位符]` 替换为您的真实信息。

### 第二步：配置个人偏好
打开 `config/profile.yml`，设置您的目标岗位、城市、薪资期望等。

### 第三步：开始使用
在 Claude 对话中，直接粘贴职位描述或链接，说：

> "帮我评估这个职位"

## 评分说明

| 等级 | 分数 | 建议 |
|------|------|------|
| A | 85-100 | 强烈推荐，优先申请 |
| B | 70-84 | 推荐申请 |
| C | 55-69 | 可以尝试 |
| D | 40-54 | 不太推荐 |
| F | 0-39 | 不推荐 |

### 评分维度
1. **岗位匹配度** (25%) — 技能与JD要求的契合程度
2. **成长空间** (20%) — 学习机会与职业发展潜力
3. **公司质量** (15%) — 品牌、规模与行业地位
4. **地点匹配** (10%) — 与偏好城市的匹配程度
5. **薪资水平** (10%) — 与期望薪资的对比
6. **经验要求匹配** (10%) — 门槛是否适合实习生
7. **工作强度与文化** (10%) — 工作环境与节奏
