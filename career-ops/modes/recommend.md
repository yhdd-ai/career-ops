# 职位推荐模式 (recommend)

## 任务
根据候选人背景和指定的求职方向，推荐一批高匹配度的公司和实习职位。

## 输入
- 候选人简历（cv.md）
- 个人偏好配置（profile.yml）
- 用户指定的求职方向（如："大模型算法"、"Java后端"、"数据分析"）

## 推荐逻辑

请综合以下因素进行推荐：
1. 候选人技能栈与方向的匹配程度
2. 公司在该方向的业务规模和技术影响力
3. 实习转正率和培养体系
4. 候选人已有背景（大厂实习经历、名校背书）能匹配的公司层级

## 输出格式

```
═══════════════════════════════════════════════════
  AI 职位推荐报告
  方向：[用户指定方向]
═══════════════════════════════════════════════════

推荐理由分析
[2-3句话说明为什么这个方向适合候选人，以及竞争优势]

═══════════════════════════════════════════════════
  推荐职位列表（共N个）
═══════════════════════════════════════════════════

【强烈推荐 ⭐⭐⭐】
─────────────────────────────────
NO.1  公司：[公司名]
      职位：[建议搜索的职位名称]
      匹配理由：[2句话说明为何匹配]
      竞争优势：[候选人哪些经历对这个岗位有优势]
      搜索建议：[去哪个平台搜、搜索关键词]
      招聘页面：[公司招聘官网链接，如已知]

NO.2  ...

【推荐 ⭐⭐】
─────────────────────────────────
NO.X  ...

【可以尝试 ⭐】
─────────────────────────────────
NO.X  ...

═══════════════════════════════════════════════════
  投递策略建议
═══════════════════════════════════════════════════
[针对该方向的投递节奏、注意事项、差异化建议]

═══════════════════════════════════════════════════
  推荐搜索关键词
═══════════════════════════════════════════════════
实习僧：[关键词1] / [关键词2]
牛客网：[关键词1] / [关键词2]
各大厂官网直投：[列出公司名和官网]
═══════════════════════════════════════════════════
```

## 注意事项
- 每个推荐等级至少 3 个，总共推荐 10-15 个职位
- 优先推荐有明确招聘页面的大厂
- 结合候选人的阿里大模型实习背景，重点推荐 AI/大模型相关岗位
- 注明哪些公司目前（4月）正处于春招/暑期实习招聘窗口
- 输出语言：中文
